计价规则：
	起步价   10元
	起步里程 2.5公里
	
	超出部分，按照计价规则收取
	--------------
	里程费   2.5元/公里
	时长费   0.5元/分钟
	远途费   1元/公里（超出15公里后，加收远途费）

	车费 = 起步价 + 里程费 + 时长费 + 远途费
	     = 起步价 + （总里程-起步里程）*每公里价 + 所用时长（分）*每分钟价 + （总里程-起步里程-远途里程）*远途每公里价 

叫车服务：
	查找周边车辆
		乘客：
			设定上车地点
			查询附近车辆
			监控附近车辆行进轨迹
				include 监控车辆行进轨迹
	呼叫约车
		乘客：
			设定行程内容
				extends 设定预约时间
			查询预估车费
			发布行程
			接收接单通知
				include 查询车主信息
				include 查询车辆信息
			监控行车过程
				include 查询行程推荐路线
				include 监控车辆行进轨迹
			查询行程车费
		车主：
			受理订单
				extends 查询订单信息
				include 暂停听单
				include 查询乘客信息
			通知乘客接单
			查询前往上车地点路线
			确认乘客上车
				include 查询行程推荐路线
			记录行车过程
				include 上传当前位置
				include 监控车辆行进轨迹
			监控车费变动
				include 查询当前计价规则
				include 计算行程车费
			确认到达
				include 结束计算行程车费
	听候订单：
		车主：
			设定听单内容
			确认听单
				include 上传当前位置
			停止听单
